MHF4U 期末项目：现实世界中的函数

游乐设施建模与函数变换

用摄影/视频与 Desmos，将九类函数的母函数做变换以匹配真实游乐设施的形状或运动，并以网页呈现。

作业对齐

这一页确保覆盖：目的、证据、分析、反思、伦理、参考文献。

可访问性

已添加语义标签、可聚焦跳转、配色对比和键盘可用性。

如何提交

可直接交单文件，或部署到 GitHub Pages/Netlify，再提交链接。

Overview – 项目简介

在此简述项目目标：应用函数变换建模真实游乐设施；使用 Desmos；制作数字网页展示。

阶段

  1. 收集照片/视频（可含模拟）
  2. 用 Desmos 建模并限定有效定义域
  3. 制作网页呈现（可自写 HTML/CSS/JS 或用建站工具）
  4. 发布并分享链接

评分对齐

知识：方程与图像正确，定义域/值域用规范记号。思维：解释行为与变换。沟通：结构清晰、无语病。展示：视觉清晰、技术运用得当。

Requirements – 必做元素

  • 共 9 类函数，至少包含：1/x、1/x^2、多项式(>2 次且多项式项)、带渐近线的三角函数、正弦/余弦、指数、对数，及 2 个自选
  • 每个函数需要：标题、游乐设施图片/视频、嵌入的 Desmos 图、变换后方程、两段文字（变换说明；定义域/值域）
  • 文字解释需为本人撰写；可用 AI 做排版与编码，不可代写数学解释

9 Functions – 模板区

每个函数使用一张卡片。先把你的照片/视频与 Desmos 链接贴进来，再写两段文字。

如何嵌入 Desmos：在 Desmos 分享中复制“Embed”代码，直接粘贴 <iframe> 到对应卡片的嵌入区即可。

Publishing – 发布你的网页

GitHub Pages

  1. 创建仓库，命名如 mhf4u-rides
  2. 将此 index.html 上传到根目录
  3. Settings → Pages → Deploy from branch

替代方案

使用 Netlify / Vercel / Google Sites / Wix / WordPress，发布后把链接交给老师。

可访问性小贴士

  • 为每张图片写 alt 文本
  • 保持颜色对比度
  • 标题层级合理

  • ✅ 已包含 9 个函数类别。
  • ✅ 每个函数均含：标题、照片/视频、Desmos 嵌入、变换后方程、两段文字。
  • ✅ 定义域/值域用规范集合或区间记号。
  • ✅ 解释为本人撰写（非 AI 代写）。
  • ✅ 页面结构清晰、语言准确、视觉整洁。
  • ✅ 网页已发布，并可通过链接访问。
